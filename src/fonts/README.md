# Custom fonts

Place your font files here. The site expects these names (or update `@font-face` in `src/index.css` to match your files):

| Use      | Font family      | Expected files                          |
|----------|------------------|----------------------------------------|
| Serif    | Chiselle Serif   | `ChiselleSerif.woff2`, `ChiselleSerif.woff` |
| Sans     | Chiselle Sans    | `ChiselleSans.woff2`, `ChiselleSans.woff`   |
| Script   | Chiselle Script  | `ChiselleScript.woff2`, `ChiselleScript.woff` |

Formats supported: `.woff2`, `.woff`, `.ttf`. Add or change the `url()` paths in `src/index.css` if your filenames differ.
